[abstract]
This is the top-level documentation entry point for the BioT Speech IoT project
(FHDW Hannover, IoT module). The project connects an ESP8266 NodeMCU microcontroller
to an Android app via MQTT. Voice commands are processed locally or forwarded to an
LLM backend for intelligent responses.

The three sub-projects are:

* `BioT_Speech_IoT_App` — Android client (MQTT subscriber, voice pipeline, live charts)
* `BioT_Speech_IoT_LLM_App` — Python FastAPI backend (LLM agent, MCP server, MQTT mirror)
* `BioT_Speech_IoT_Doc` — this documentation project
